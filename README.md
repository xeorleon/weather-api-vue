# Weather Api

### Tech


* [Vue.js](https://vuejs.org/) 
* [Node.js](https://nodejs.org/)
* [OpenWeatherAPI](https://openweathermap.org/)



### Installation

Install the dependencies and start the server.

```sh

$ npm install
$ npm run serve
```
